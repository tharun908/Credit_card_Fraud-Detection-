# Credit Card Fraud Detection

## Overview

This project uses Machine Learning to detect fraudulent credit card transactions. A Logistic Regression model was trained on historical transaction data and deployed as an interactive Streamlit web application.

## Features

* Data preprocessing and cleaning
* Exploratory Data Analysis (EDA)
* Logistic Regression model
* Fraud prediction system
* Streamlit web application
* GitHub version control

## Technologies Used

* Python
* Pandas
* NumPy
* Scikit-learn
* Streamlit
* Pickle
